# cd

## Descripción

Se utiliza para cambiar entre directorios/carpetas dentro de la terminal.

---

## Sintaxis (como se utiliza el comando "cd" en la terminal)

cd [ruta_directorio_/_opcion]

---

## Opciones útiles

### ..
Permite retroceder al directorio anterior.

### ~
Lleva directamente al directorio personal del usuario (/home/"usuario").

### /
Lleva al directorio raíz del sistema.

---

## Ejemplos

Entrar a una carpeta: cd documentos

Retroceder una carpeta: cd ..

Ir al directorio personal: cd ~

---

## Errores y problemas comunes

#### Error:
Intentar acceder a una carpeta que no existe.

-Ejemplo incorrecto: cd tareas

#### Solución:
Verificar primero el nombre del directorio utilizando "ls".

-Ejemplo correcto 1: ls
-Ejemplo correcto 2: cd documentos
