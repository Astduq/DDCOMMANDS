# ip

## Descripción

Se utiliza para administrar interfaces y configuraciones de red en Linux.

Permite consultar direcciones IP, rutas y estado de interfaces.

---

## Sintaxis (como se utiliza el comando "ip" en la terminal)

ip [opción]

## Opciones útiles

### a
Muestra direcciones IP de las interfaces.

### route
Muestra rutas de red.

### link
Muestra interfaces disponibles.

---

## Ejemplos

Ver direcciones IP: ip a

Mostrar rutas de red: ip route

Mostrar interfaces: ip link

---

## Errores y problemas comunes

#### Error:
Confundir opciones del comando.

-Ejemplo incorrecto: ip -a

#### Solución:
Utilizar las opciones correctas del comando ip.

-Ejemplo correcto 1: ip a
-Ejemplo correcto 2: ip route
