# DDCOMMANDS

Herramienta educativa para aprender comandos Linux desde terminal.

## Características

- Explicaciones simples
- Ejemplos reales
- Errores comunes
- Uso offline
- Compatible con Debian

## Instalación

sudo dpkg -i ddcommands.deb

## Dependencias

sudo apt install glow

## Uso:

ddcommands "comando"

## Tecnologías usadas

- Bash
- Markdown
- Linux
- Glow
