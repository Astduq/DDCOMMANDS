# DDCOMMANDS

Herramienta educativa para aprender comandos Linux desde terminal.

## Características

- Explicaciones simples
- Ejemplos reales
- Errores comunes
- Uso offline
- Compatible con Debian

## Instalación

sudo dpkg -i ddcommands_1.0.deb

## Dependencias NO obligatorias pero recomendadas

sudo apt install glow

## Uso:

ddcommands "comando"

## Tecnologías utilizadas

- Bash
- Markdown
- Linux
- Glow
- Github
