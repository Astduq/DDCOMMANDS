# uname

## Descripción

Se utiliza para mostrar información del sistema operativo y del kernel.

Es útil para conocer detalles técnicos del sistema Linux.

---

## Sintaxis (como se utiliza el comando "uname" en la terminal)

uname [opciones]

## Opciones útiles

### -a
Muestra toda la información disponible.

### -r
Muestra versión del kernel.

### -m
Muestra arquitectura del sistema.

---

## Ejemplos

Ver información básica: uname

Ver toda la información: uname -a

Ver arquitectura: uname -m

---

## Errores y problemas comunes

#### Error:
Esperar información demasiado detallada.

-Ejemplo incorrecto: uname (Buscar información específica de hardware)

#### Solución:
Usar opciones más completas o combinar con otros comandos.

-Ejemplo correcto 1: uname -a
-Ejemplo correcto 2: lsblk
